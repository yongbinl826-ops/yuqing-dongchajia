========================================
快速开始
========================================

第一步：安装依赖

Windows用户：
  右键点击 install_windows.bat，选择"以管理员身份运行"

Linux/Mac用户：
  在终端运行: bash install_linux.sh

第二步：配置数据库
  参考 操作说明.md 中的"配置数据库"章节

第三步：启动应用
  python3 nlp_service.py
  PORT=3001 pnpm start
  访问: http://localhost:3001

详细说明请参考 操作说明.md

========================================
系统要求
========================================

最低配置：
- 操作系统：Windows 10/11 或 Linux (Ubuntu 20.04+)
- CPU：4核心及以上
- 内存：8GB RAM
- 硬盘：10GB可用空间
- Node.js：v18.0.0+
- Python：3.9+
- MySQL：8.0+

========================================
项目特点
========================================

1. AI技术应用
   - 基于Transformers的情感分析
   - 中英文双语支持
   - 实时NLP处理

2. 全栈开发
   - 现代化前端框架（React + TypeScript）
   - 高效后端API（Node.js + tRPC）
   - 强大数据库（MySQL + Drizzle ORM）

3. 多平台支持
   - Twitter、Reddit、YouTube等平台数据采集
   - 统一的数据模型
   - 灵活的扩展机制

4. 完整功能
   - 舆情监控和采集
   - 情感分析和分类
   - 数据可视化展示
   - 结果导出和分享

========================================
核心功能
========================================

1. 监控任务管理 - 创建和管理监控任务
2. 数据采集 - 自动采集相关内容
3. 情感分析 - 自动分析内容情感
4. 数据分析 - 情感分布、趋势分析
5. 可视化展示 - Dashboard、图表展示

========================================
技术栈
========================================

前端：React 18+、TypeScript、Vite、TailwindCSS
后端：Node.js、Express、tRPC、Drizzle ORM
数据库：MySQL 8.0+
AI/ML：Python 3.9+、FastAPI、Transformers、PyTorch

========================================
项目结构
========================================

源代码/
├── client/                 # 前端代码
├── server/                # 后端代码
│   ├── _core/            # 核心服务
│   ├── routers/          # API路由
│   ├── collectors/       # 数据采集器
│   ├── db.ts             # 数据库操作
│   └── nlp_service.py    # NLP服务
├── drizzle/              # 数据库schema
├── package.json          # Node.js配置
├── requirements.txt      # Python依赖
└── README.md             # 项目说明

========================================
使用说明
========================================

1. 创建监控任务 - 点击"创建新任务"，输入关键词，选择数据源
2. 开始采集 - 进入任务详情，点击"开始采集"
3. 查看结果 - 查看采集数据和分析结果
4. 导出数据 - 点击"导出"下载数据文件

详细说明请参考 操作说明.md
